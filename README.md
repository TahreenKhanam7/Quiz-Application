# Java Quiz Application

A simple Java Swing-based quiz application built using Apache NetBeans.

## 🚀 Features

- 10-question multiple choice quiz
- Timer for each question
- Score calculation at the end

## 🛠️ Technologies Used

- Java
- Swing
- Apache NetBeans
